UCRAPI.JUSTINF.DEV

A JSON API providing registration and course data for the University of California, Riverside.

This API sources public data from UCR's Ellucian Banner, updating every 12 hours.

The main purpose of this API is to allow applications that use UCR class data to access up-to-date information easily.
